## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Creating Application

- [Nest CLI](https://docs.nestjs.com/cli/overview)를 사용하여 새 프로젝트를 생성합니다.

```bash
$ nest g application uber-eats-backend
```

## Installation

```bash
$ cd uber-eats-backend
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
